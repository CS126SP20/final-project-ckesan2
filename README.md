**Quick Click**:

* What is it? 

   - It is a game for users to click on moving objects for
    points with a time limit. 
    
   - Includes three different game modes 
   
   - Keeps a leader board for each difficulty
 
   - Makes use of the Cinder, SQLLiteModernCpp, and 
   SQLLite3 libraries
